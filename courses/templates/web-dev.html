Web Developement

Duration : 4 Months
Eligiblity : 10th Passed

Web development refers to the creating, building, and maintaining of websites. It includes aspects such as web design, web publishing, web programming, and database management. It is the creation of an application that works over the internet i.e. websites.

The word Web Development is made up of two words, that is :

  • Web : It refers to websites, web pages or anything that works over the internet.
  • Development :It refers to building the application from scratch.

Syllabus

  • Basics of HTML, CSS and JS
  • Page Layout Basics
  • Browser Compatibility
  • Test on all popular browsers
  • Navigation
  • Color and Graphics
  • Multimedia
  • Content
  • Use common fonts throughout and be consistent in size and color
  • Use headings, bullet points and brief paragraphs
  • SVGs
  • And many other things

Web Developement contains the two part Frontend and Backend. The Frontend part contains HTML, CSS and JS, on the other hand the Backend part contains Server side scripting with a programming language like Python, C language, JavaScript and many others.
