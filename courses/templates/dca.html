DCA - Diploma in Computer Application

Duration : 6 Months
Eligiblity : 10th Passed
Diploma Program for Graphic Designer and Web Designer.

Diploma in Computer Application is course to build career as a Teacher/Banker/ Designer of information technology.

Syllabus

  • Computer Fundamentals
  • Operating System and Troubleshooting
  • Microsoft Office with Access and Outlook
  • Internet Technology and Security
  • Website Designing
  • Programming Logic & Techniques Using C
  • Communication and Presentation Skills

Benefits

  • Helps in clearing easily, the written test conducted by IBPS for recruitment of clerks/officers in Banks.
  • About 40/50 questions (15% of the total marks) are based on computer.
  • Help in Working efficiently offer (selection) in banks recruitments.
